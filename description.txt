2. Краткое описание сюжета
Candy Dream, игра-платформер, в которой главному персонажу сниться сон, что он попал в мир конфет,
в котором его главной целью становится найти ключ к двери, с помощью которой он может попасть обратно в мир людей.
переход между локациями будет осуществляться по мере сбора ключей, находящихся в конце каждой из локаций.

3. Детальное описание персонажа
Главный герой - серый кот. Не имеет особых навыков боя, который должен пройти определенные испытания

4. Геймплей

Игра состоит из определенных глав, делящихся по локациям. На каждой локации появляются определенные враги
и второстепенные персонажи, которые могут давать подсказки. В процессе игры, игрок собирает очки.
Управление котом осуществляется с помощью клавиатуры.
